Question-1: What are the different ways to select an element in the DOM?


  • document.getElementById('id')
  • document.getElementsByClassName('class')
  • document.getElementsByTagName('tag')
  • document.querySelector('selector')
  • document.querySelectorAll('selector')

Question-2: What is the difference between innerHTML, innerText, and textContent?


  • innerHTML: Gets the HTML & text content inside an element, including any HTML tags.
  • innerText: Gets the text content inside of an element.
  • textContent: Gets the text content of an element, but excluding HTML tags.

Question-3: What is event delegation in the DOM?


Event delegation is a technique where add a single event listener to a parent element, which can handle events for all of its child elements.

Question-4: What is event bubbling in the DOM?


Event bubbling is when an event on a child element moves up to its parent elements, allowing them to handle it.

Question-5: How do you create, add, and remove elements using JavaScript?


  • Create: Use document.createElement('tagName') to create a new element.
  • Add: Use parent.appendChild(element) to add it to the DOM.
  • Remove: Use element.remove() or parent.removeChild(element) to delete an element..
